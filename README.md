# 🐾 Petora - Pet Adoption Platform

## 📌 Project Purpose

Petora is a modern pet adoption platform where users can explore pets, send adoption requests, and manage their pet listings easily. The platform provides a smooth and responsive experience for both pet owners and adopters.

## ✨ Features

- 🔍 Search pets by name using MongoDB `$regex`
- 🐶 Filter pets by species
- 📊 Sort pets by adoption fee
- 🔐 JWT Authentication & Protected Routes
- 📝 Add, Update, and Delete Pet Listings
- ❤️ Send and Manage Adoption Requests
- ✅ Approve or Reject Adoption Requests
- 📱 Fully Responsive Modern UI
- 🌙 Beautiful UI with HeroUI Components
- ⚡ Fast and Optimized Next.js App Router Structure
